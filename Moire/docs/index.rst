Latticegen
==========

.. image:: https://zenodo.org/badge/DOI/10.5281/zenodo.5156831.svg
   :target: https://doi.org/10.5281/zenodo.5156831
.. image:: https://readthedocs.org/projects/moire-lattice-generator/badge/?version=latest
   :target: https://moire-lattice-generator.readthedocs.io/en/latest/?badge=latest
   :alt: Documentation Status
.. image:: https://img.shields.io/pypi/v/latticegen
   :target: https://pypi.org/project/latticegen/
   :alt: PyPI

Simple python code to interactively generate visualizations of moire patterns of lattices, such as for example magic angle bilayer graphene. 


Installation
------------

.. code-block:: bash

   pip install latticegen
